

* Commandes d'exécution pour lancer le modele via la console ACT-R :

(place-valises 1): execute le modele une seul fois (argument 1). Cette commande n'affiche pas le schéma du coffre avec les valises.

(place-valises 1 t): execute le modele une seul fois et affiche l'organisation des valises dans le coffre.

(show-learning 20): execute 20 blocs de 1000 et affiche le graphe d'apprentissage.


- les valeurs numériques des paramétres peuvent être changées, selon le besoin.
